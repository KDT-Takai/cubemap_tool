# Equirectangular Å® Cubemap Dice Layout Tool

This tool converts an equirectangular image into a cubemap and generates a dice layout.

## Usage

1. Run the program
2. Enter input image absolute path
3. Enter face size (default 512)
4. Choose dice layout:
   - [1] Cross (Horizontal)
   - [2] Cross (Vertical)
   - [3] Horizontal strip
   - [4] Vertical strip
5. Handle aspect ratio prompt if needed
6. Wait for conversion and output
